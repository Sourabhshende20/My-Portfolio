Anti-Sleep Alert System

A hardware prototype built on Arduino UNO that detects eye closure using an IR sensor mounted on safety glasses. If the eyes remain closed for more than 3 seconds, the system triggers a buzzer and a blinking red LED alert — simulating a real-world drowsiness detection mechanism for safety applications.

  • IR sensor detects eye state in real time; alert triggers after 3 seconds of continuous closure
  • Outputs a buzzer alarm and blinking red LED upon detection
  • Supports Serial Monitor debugging and optional cloud-based alert integration

Li-Fi

Li-Fi Data Transmission System

A working Li-Fi prototype that transmits data wirelessly using modulated light instead of radio waves. The transmitter PCB — built on a custom yellow board — takes audio input via a microphone or aux cable, amplifies it, and modulates an LED's brightness to carry the signal. The receiver uses a solar panel as a photodetector, converting the light variations back into electrical current, which is amplified and played through a speaker.

  • Achieves wireless data transmission over 10–15 feet using light as the communication medium
  • Transmitter features a custom PCB with battery, audio amplifier, mic/aux input, and modulated LED
  • Receiver uses a solar panel (wider detection area than LDRs) feeding an audio amplifier and speaker
